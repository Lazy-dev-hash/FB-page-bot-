🚀 Quick Setup Guide

1

🔐 Configure Your API Keys

Set up your educational bot credentials in Replit Secrets:

  • PAGE_ACCESS_TOKEN - Your Facebook Page Access Token
  • VERIFY_TOKEN - Custom webhook verification token
  • OPENAI_API_KEY - Your OpenAI GPT-4 API key
2

🌐 Deploy Your EduBot

Deploy this Replit project to get a live educational assistant that can help students learn!

3

🔗 Connect Facebook Webhook

In your Facebook Developer Console, configure:

  • Webhook URL: https://your-repl-url.com/webhook
  • Verify Token: Your VERIFY_TOKEN value
  • Subscribe to: messages events
